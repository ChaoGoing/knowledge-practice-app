# 基于 pnpm + vite + electron 的多功能应用软件


1. 首先创建项目目录并初始化
    ```bash 
    mkdir my-app && cd my-app
    ```
2. 安装 pnpm 包管理工具
    ```bash
    npm install -g pnpm
    ```
3. 安装 vite 构建工具
    ```bash
    pnpm create vite
    ```
4. 安装 electron 框架
    ```bash
    pnpm add electron --save-dev
    ```
5. 安装 electron-builder 打包工具
    ```bash
    pnpm add electron-builder --save-dev
    ```
